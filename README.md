## 데이터 전처리 과정 (Data Preparation)

분석의 신뢰도를 높이기 위해 파이썬(Pandas)과 Tableau Prep을 병행하여 데이터 정제를 진행함. (흩어져 있는 원천 데이터를 분석이 가능한 '하나의 진실된 데이터(Single Source of Truth)'로 만드는 핵심 단계)

### ① 데이터 통합 및 구조화 (Data Merging)
* 여러 소스의 결합: 고객의 프로필(`portfolio`), 이벤트 수신 기록(`profile`), 그리고 실제 트랜잭션 데이터(`transcript`) 등 서로 다른 출처의 CSV 파일들을 공통 ID를 기준으로 병합했음.
* 비정형 데이터 처리: 텍스트(Dictionary) 형태로 저장된 이벤트 데이터를 파이썬을 활용해 각각의 열(Column)로 분리하여, 태블로에서 즉시 분석할 수 있는 정형 데이터 형태로 변환했음.

### ② 데이터 정제 및 이상치 처리 (Data Cleaning)
* 결측값 및 오류 해결: 나이(Age) 항목에 포함된 이상치(예: 118세 등 기록 오류)와 성별 정보가 누락된 데이터를 식별하고, 분석 목적에 맞게 제거하거나 평균값으로 대체하여 왜곡을 방지했음.
* 표준화 작업: 서로 다른 단위의 수치 데이터들을 비교 가능하도록 표준화(StandardScaler)하고, 날짜 및 시간 데이터를 분석에 용이한 형식으로 재설정했음.

### ③ 분석용 파생 변수 생성 (Feature Engineering)
* 고객 세분화: 가입 기간, 총 구매 금액, 방문 빈도를 계산하여 고객을 그룹화(RFM 분석 기반)할 수 있는 지표를 생성했음.
* 영양 성분 범주화: 메뉴별 칼로리와 당류 함량을 기준으로 '고/중/저' 그룹을 나누어, 시각화 시 직관적인 비교가 가능하도록 설정했음.
