# 스타벅스 프로모션 성과분석: 세그먼트별 마케팅 전략
- 본 프로젝트는 Python에 더하여, Tableau 및 Tableau Prep도 활용하여 데이터 전처리 및 결합, 데이터 EDA를 진행해 본 프로젝트임.
- Python과 Tableau(Tableau Prep)을 교차 검증하여 더욱 정교한 데이터 전처리 및 결합과 데이터 EDA 결과를 얻고자 함.

## 프로젝트 요약
스타벅스의 고객 그룹을 식별하고, 데이터를 기반으로 그룹별 프로모션 효과를 분석

## 프로젝트 목표
1. 고객 그룹을 식별하고, 데이터를 기반으로 프로모션 효과를 분석
2. Tableau 대시보드를 제작하여 분석 결과를 직관적으로 표현하고, 이해관계자에게 전달 가능한 시각화 스토리텔링 능력 강화
3. KPI, 고객 세그먼트별 반응률, 채널별 성과 등을 시각화 하여 데이터 기반의 의사결정을 지원하는 대시보드 제작 역량 강화
